## Tech Stack

**Client:** React, Typescript, TailwindCSS and Crypress

## Features - TodoList

- Create Todo
- Completed Todo
- Filter Todo in all, active and completed
- Clear completed Todos
- Delete specific todo


## Running Tests

To run tests, run the following command

```bash
  yarn cypress:run
```
or 

```bash
  npx run cypress:run
```
